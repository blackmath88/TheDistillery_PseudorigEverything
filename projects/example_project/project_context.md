# Project Context

This project is about building a local knowledge distillation pipeline called The Distillery.

The goal is to take messy human inputs — transcripts, notes, PDFs, workbooks — and transform them
into structured, AI-ready knowledge artefacts.

## Background

The creator has been accumulating raw notes, voice transcripts, and reference materials for months.
This project aims to make that corpus searchable, summarised, and ready for downstream AI tools.

## Key Themes

- Local-first processing (no cloud required)
- Modular pipeline stages (extract → clean → chunk → summarise)
- Support for multiple file types: txt, md, pdf
- Output formats: JSON, JSONL, Markdown
